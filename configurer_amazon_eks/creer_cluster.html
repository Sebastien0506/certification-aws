Préparer l'environement aws Kubernetes
Avant de créer votre premier cluster,
il faut s'assurer d'avoir installé tous les outils requis sur le système que vous utiliserez pour accéder au cluster:
  • AWS Command Line Interface (AWS CLI)
  • kubectl
  • (facultatif) eksctl

il est important de noter que la création d'un cluster fait référence au processus de déploiement du plan de contrôle géré par Amazon EKS.

UTILITAIRE EKSCTL:

eksctl est un outil en ligne de commande qui simplifie la création de cluster. En utilisant cette méthode, on peut créer un cluster à l'aide d'une seule commande.
